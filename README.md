Multi Panel
===========

Author: inasis

Extension inspired by https://github.com/darkxst/multiple-monitor-panels 
and rewritten from scratch for gnome-shell version 3.10.4.  Adds panels 
and thumbnails for additional monitors. Settings changes are applied 
in dynamic fashion, no restart needed.

Versions
========

* Branch [master](https://github.com/inasis/multi-panel/tree/master) contains extension for GNOME 3.38
* Branch [gnome-3-32_3-36](https://github.com/inasis/multi-panel/tree/gnome-3-32_3-36) contains extension for GNOME 3.32, 3.34 and 3.36
* Branch [gnome-3-24_3-30](https://github.com/inasis/multi-panel/tree/gnome-3-24_3-30) contains extension for GNOME 3.24, 3.26, 3.28 and 3.30
* Branch [gnome-3-20_3-22](https://github.com/inasis/multi-panel/tree/gnome-3-20_3-22) contains extension for GNOME 3.20 and 3.22
* Branch [gnome-3-16_3-18](https://github.com/inasis/multi-panel/tree/gnome-3-16_3-18) contains extension for GNOME 3.16 and 3.18
* Branch [gnome-3-14](https://github.com/inasis/multi-panel/tree/gnome-3-14) contains extension for GNOME 3.14
* Branch [gnome-3-10](https://github.com/inasis/multi-panel/tree/gnome-3-10) contains extension for GNOME 3.10

Installation from git
=====================

    git clone https://github.com/inasis/multi-panel
    cd multi-panel
    cp -r multi-panel@inasis ~/.local/share/gnome-shell/extensions/

Restart the shell and then enable the extension.
